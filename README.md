# Yasar Ameen Portfolio

A responsive 3D portfolio website for Yasar Ameen H N, an Agentic AI Developer and Full-Stack Engineer. The site highlights AI projects, technical skills, internship experience, certifications, and contact links.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- React Three Fiber
- Drei
- Three.js

## Features

- Single-page portfolio layout
- Interactive 3D hero scene
- Resume-backed content
- Animated section backgrounds
- Responsive design for mobile, tablet, and desktop
- Downloadable resume at `/resume.pdf`

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Project Structure

```text
app/                  Next.js app router files
components/           UI, section, provider, and 3D components
lib/                  Portfolio content and data
public/               Static assets, including resume.pdf
resume/               Source resume file
docs/                 Planning, design, content, and implementation docs
```
